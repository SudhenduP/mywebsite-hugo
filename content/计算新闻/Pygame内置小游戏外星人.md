## 安装pygame
首先尝试：
> pip install pygame -i http://pypi.douban.com/simple

如果出错，尝试
> pip install pygame -i http://pypi.douban.com/simple --trusted-host pypi.douban.com

## 开始玩游戏！

[https://www.pygame.org/](https://www.pygame.org/)

> python3 -m pygame.examples.aliens
